2. Your Project

  1. Before you can get started rendering maps, using services, and generally being a really cool individual, you need to have your own Google Developer Project.

    If you don’t already have one, you can go to the Google APIs Console and create your own project. This is free and will allow you to generate your own API key to use within your site.

    1. 1. Visit https://console.developers.google.com

    2. 2. Sign in with your Google - i.e. yourname@gmail.com.

    3. 3. Click on "Select a Project" and then "Create a Project" in the upper right side of the screen.

    4. 4. Name your project something descriptive, such as "Udacity-Maps-Project" or "Emily-likes-buffalo-wings". Select whether you'd like to receive updates (usually a good idea) and select whether you agree to the terms of service.

    5. 5. Click the "Create" button.
    6. 6. Click on the "Google Maps JavaScript API" under "Google Maps APIs"
    7. 7. Click the "Enable" button
    8. 8. Repeat steps 6 and 7 for the following APIs (highlighted in yellow) which we'll use in this course:
      1. Google Maps JS API
      2. Google Maps Roads API
      3. Google Static Maps API
      4. Google Street View Image API
      5. Google Places API Web Service
      6. Google Maps Geocoding API
      7. Google Maps Directions API
      8. Google Maps Distance Matrix API
      9. Google Maps Geolocation API
      10. Google Maps Elevation API
      11. Google Maps Time Zone API
    9. 9. Click the “Credentials” menu item on the left of the screen
    10. 10. Click the “Create Credentials” button
    11. 11. Select “API Key” from the options that appear.
    12. 12. Select the “Browser Key” button
    13. 13. Name your API Key something descriptive, such as Maps_API_Udacity_Course. It should reflect your intended use of the key. Click the “Create” button
    14. 14. Repeat steps 10 and 11 for “Server Key,” if you will be making server-side requests. You can access these keys at any time by visiting the “Credentials” sub-menu under the “API Manager” menu
    15. 15. Get mapping!
